# User Stories

## Theme: E-Commerce & Payments
*   **US-01:** As a construction professional, I want to purchase a CAD extension using BLIK, so that the transaction is fast, mobile-friendly, and secure.
*   **US-02:** As an international architect, I want to pay for software licenses via PayPal, so I can use my trusted and preferred payment method.
*   **US-03:** As a corporate buyer, I want to pay via secure direct bank login, so the transfer is processed immediately and securely.

## Theme: Educational Content & Publishing
*   **US-04:** As an industry expert (Author), I want to quickly publish articles about the latest construction tech discoveries using a secure WYSIWYG editor without risking platform security.
*   **US-05:** As a student/trainee, I want to browse and purchase premium educational modules about CAD software, so I can improve my industry skills.
*   **US-06:** As a user, I want the website to load instantly on my mobile device, so I can read about new software discoveries on the construction site.

## Theme: Security & Administration (SSDLC)
*   **US-07:** As a system administrator, I want to require MFA for all backend logins, so that the platform is protected against credential stuffing and brute-force attacks.
*   **US-08:** As a security auditor, I want all payment data to be handled off-site by the payment gateway, so that our MySQL database remains out of PCI-DSS scope.
*   **US-09:** As a site manager, I want automated backups of the MySQL database and WP files to occur daily, so we can recover quickly in the event of a ransomware attack or failure.
