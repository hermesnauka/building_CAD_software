# Project: Construction Tech & CAD Edu-Commerce Platform

## Overview
This project is a WordPress-based educational and e-commerce platform focused on the latest discoveries in construction software and CAD extensions.
It utilizes a MySQL database and integrates with major payment gateways (BLIK, PayPal, Bank Transfers/Logins like PayU/Przelewy24).
The development and maintenance processes strictly follow Secure Software Development Life Cycle (SSDLC) principles to ensure data protection and secure financial transactions.

## AI Context & Instructions
When assisting with this project, strictly adhere to WordPress security best practices (e.g., OWASP Top 10 for WordPress) and SSDLC guidelines.
Always assume that payment data tokenization and secure database queries (prepared statements) are mandatory.
